# Lexa AI

**Your Premium AI Assistant**

A next-generation AI chat interface with stunning visual design, advanced features, and seamless user experience.

## Features

- **Multi-model AI** - Switch between different AI models for various use cases
- **Voice Chat** - Real-time voice conversations with AI
- **Image Generation** - Create stunning images with AI
- **Video Generation** - Generate videos using Veo 3.1
- **Deep Research** - Comprehensive AI-powered research
- **Knowledge Base** - Personal knowledge management
- **Goals & Tasks** - AI-assisted productivity
- **Focus Mode** - Distraction-free work sessions
- **Personalization** - Customizable AI personality and preferences

## Tech Stack

- **React 18** - Modern UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Premium animations
- **shadcn/ui** - Beautiful UI components
- **Supabase** - Backend & Authentication

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Development

```bash
# Run tests
npm run test

# Lint code
npm run lint
```

## Project Structure

```
src/
├── components/     # UI components
│   ├── auth/       # Authentication components
│   ├── chat/       # Chat interface components
│   └── ui/         # Base UI primitives
├── hooks/          # Custom React hooks
├── pages/          # Page components
├── lib/            # Utilities
└── types/          # TypeScript types
```
